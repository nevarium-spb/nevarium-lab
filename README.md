# Невариум ЛАБ ИИ

Многостраничный SaaS-сайт лаборатории внедрения ИИ в бизнес. React + TypeScript + Vite,
дизайн-система «Vizor» в стиле глассморфизм, встроенный чат-бот «Нева» для общения с клиентами.

## Запуск

```bash
npm install
npm run dev        # дев-сервер на http://localhost:5173
npm run build      # production-сборка в dist/
npm run preview    # предпросмотр production-сборки
```

## Структура

| Путь | Что это |
|---|---|
| `src/pages/` | Страницы: Главная, Услуги, Кейсы, Тарифы, Лаборатория, Контакты, 404 |
| `src/components/` | Хедер, футер, чат-бот, aurora-фон, scroll-reveal, счётчики, hero-визуализация |
| `src/data/content.tsx` | Весь контент сайта: услуги, кейсы, тарифы, FAQ — правится в одном месте |
| `src/data/bot.ts` | Сценарий чат-бота: дерево узлов + маршрутизация свободного ввода |
| `src/styles/global.css` | Дизайн-система: токены, глассморфизм, анимации, адаптив |
| `docs/DESIGN_DOC.md` | Проектный документ: позиционирование, архитектура, тест-план |

## Чат-бот «Нева»

Сценарный ассистент без бэкенда: ветки «услуги / цены / как начать», сбор лида
(имя + контакт), понимание свободного текста по ключевым словам, история диалога
в sessionStorage. Точка расширения: подключить реальный LLM-бэкенд в `Chatbot.tsx`
(функция `speak`) и отправку лидов в CRM.

## Куда подключить бэкенд

- **Форма контактов** — `src/pages/Contact.tsx`, обработчик `onSubmit` (сейчас демо-режим).
- **Лиды из чат-бота** — `src/components/Chatbot.tsx`, узел `lead_done`.

## Деплой

Статический сайт: содержимое `dist/` можно выложить на любой хостинг
(Netlify, Vercel, GitHub Pages, VPS + nginx). Для SPA-роутинга настройте
fallback всех маршрутов на `index.html`.
