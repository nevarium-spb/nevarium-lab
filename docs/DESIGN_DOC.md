# Невариум ЛАБ ИИ — Design Doc (Think → Plan)

## 1. Office Hours — шесть вопросов

**Кто клиент?** Малый и средний бизнес в РФ/СНГ (e-commerce, услуги, онлайн-школы, B2B-продажи), у которого нет своей команды по ИИ, но есть рутина: контент, ответы клиентам, отчёты, лидогенерация.

**Какую боль решаем?** «Слышал про нейросети, но не понимаю, с чего начать и что это даст в деньгах». Продаём не технологии, а результат: часы, снятые с сотрудников, и рост конверсии.

**Почему мы?** Лаборатория одного эксперта-практика: системный подход, каскадные промпты, готовые GPT-ассистенты и шаблоны для 15+ ниш (HR, e-commerce, маркетплейсы, лидогенерация, копирайтинг, SMM). Личная работа без конвейера агентства.

**Что 10-звёздочный продукт?** Сайт, который сам демонстрирует продукт: посетитель общается с ИИ-ассистентом «Нева» прямо на сайте и видит, как это работало бы у него.

**Минимальный сильный скоуп?** 6 страниц + чат-бот. Никакого блога и личных кабинетов на старте.

**Метрика успеха?** Заявка (лид) через чат-бот или форму контакта.

## 2. Позиционирование

- Название: **Невариум ЛАБ ИИ** (Nevarium LAB AI)
- Слоган: «Внедряем ИИ в бизнес. Измеримо.»
- Тон: уверенный, конкретный, без хайпа. Цифры вместо прилагательных.

## 3. Услуги (из реальных компетенций владельца)

1. **GPT-ассистенты под ключ** — системные инструкции, tone of voice, ролевые и каскадные промпты.
2. **Чат-боты для продаж и поддержки** — сценарии, квалификация лидов, автоматизация (SendPulse).
3. **ИИ-контент и маркетинг** — копирайтинг, SMM, сценарии Reels, УТП, воронки.
4. **ИИ-таблицы (GPT Sheets)** — каскады из 10–100 промптов: анализ ЦА, HR, e-commerce, лидогенерация.
5. **Генерация визуала и видео** — Midjourney, Flux, Kandinsky, Kling; дизайн, логотипы, ролики.
6. **Обучение команды** — промпт-инжиниринг: 16 техник, библиотека ассистентов.

## 5. Архитектура

- **Стек:** Vite + React 18 + TypeScript + react-router-dom. Zero UI-библиотек — свой дизайн-код.
- **Стили:** чистый CSS, дизайн-токены в `:root`, глассморфизм-утилиты.
- **Анимации:** CSS keyframes (aurora-фон, свечение), IntersectionObserver-хук `useReveal` для scroll-reveal, счётчики.
- **Изображения:** инлайн-SVG (нейросеть-граф, мокапы дашбордов) — быстро, резко на retina, без внешних зависимостей.
- **Чат-бот «Нева»:** клиентский сценарный ассистент: быстрые ответы, ветки (услуги/цены/как начать), сбор лида (имя + телефон), typing-индикатор, хранение диалога в sessionStorage.
- **SEO:** `lang="ru"`, title/description на страницу, семантическая разметка, Open Graph.
- **A11y:** контраст ≥ 4.5:1 для текста, focus-visible, aria-метки, prefers-reduced-motion.

## 6. Дизайн-система «Nevarium Vizor» (новая, без старых стилей)

- **Настроение:** глубокий космос лаборатории: тёмный фон #07080F, aurora-градиенты (фиолет → циан → мажента), стеклянные панели.
- **Глассморфизм:** `backdrop-filter: blur(20px)`, фон `rgba(255,255,255,0.04–0.08)`, бордер `rgba(255,255,255,0.1)`, внутренний блик сверху.
- **Акценты:** электрический циан `#22d3ee`, виолет `#8b5cf6`, мажента `#e879f9`.
- **Типографика:** Unbounded (заголовки, кириллица) + Manrope (текст). Fluid-размеры через `clamp()`.
- **Сетка:** контейнер 1200px, брейкпоинты 1024/768/480.
- **Мобильная версия:** burger → стеклянная шторка, стек-раскладка, чат-бот на весь экран.

5. `npm run build` проходит без ошибок.
